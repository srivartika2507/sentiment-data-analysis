# Sentiment Data AnalysiS

## Overview
This project demonstrates basic data analysis skills including data cleaning,
exploratory data analysis (EDA), and basic data visualization using Python.

## Tasks Completed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Basic data visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Folder Structure
- data/ : Raw and cleaned datasets  
- sentiment_cleaning.ipynb : Jupyter notebook with data cleaning, EDA, and visualization

- visuals/ : Generated plots and charts  

## Key Learnings
- Handling missing values and duplicates
- Converting timestamps to datetime
- Visualizing data using different plot types
